|                                | Obsidian Markdown                                                                                           | Alternative                            | HTML tags                                                                      |
| ------------------------------ | ----------------------------------------------------------------------------------------------------------- | -------------------------------------- | ------------------------------------------------------------------------------ |
| <h4>Inline text</h4>           |                                                                                                             |                                        |                                                                                |
| Links<br>                      | `[[href\|text]]`<br><br>`[text](href)`<br><br>`http://...`<br>`<http://...>`<br><br>`**@**.**`              |                                        | `<a href="...">...</a>`                                                        |
| Paragraphs                     | `...\s\s\n...`<br>`...\n\n...`                                                                              |                                        | `<p>...`<br>`<p>...`                                                           |
| *italic*                       | `*...*`<br><br>`_..._`                                                                                      | No `_..._`                             | `<em>...`                                                                      |
| **bold**                       | `**...**`<br><br>`__...__`                                                                                  | No `__...__`                           | `<strong>...`                                                                  |
| ***bold & italic***            | `***...***`<br><br>`___...___`                                                                              | No `___...___`                         | `<strong><em>...`                                                              |
| ~~strikthrough~~               | `~~...~~`                                                                                                   |                                        | `<s>...`                                                                       |
| `code`                         | `` `...` ``                                                                                                 |                                        | `<code>`                                                                       |
| $x \in \mathbb R$              | `$...$`                                                                                                     |                                        | Katex MathJax                                                                  |
| ==Highlight==                  | `==...==`<br><br>`=={red}...==`                                                                             |                                        | `<mark>`                                                                       |
| <u>underline</u>               |                                                                                                             | `__...__`                              | `<u>...`                                                                       |
| <sup>superscript</sup>         |                                                                                                             | `^...^`                                | `<sup>`                                                                        |
| <sub>subscript</sub>           |                                                                                                             | `~...~`                                | `<sub>`                                                                        |
| <h4>Text content</h4>          |                                                                                                             |                                        |                                                                                |
| Headings                       | `# Heading1`<br>`## Heading2`<br>`### Heading3`<br>`#### Heading4`<br>`##### Heading5`<br>`###### Heading6` |                                        | `<h1>...`<br>`<h2>...`<br>`<h3>...`<br>`<h4>...`<br>`<h5>...`<br>`<h6>...`<br> |
| Line Feed                      | `ŌÉŻŌÉŻ` or `\n`                                                                                                 |                                        | `<br>`                                                                         |
| <hr>                           | `***`<br>`---`<br>`___` (underscores)                                                                       | No `***`                               | `<hr>`                                                                         |
| Unordered Lists                | `- a`<br>`* b`                                                                                              |                                        | `<ul><li>...`                                                                  |
| Ordered Lists                  | `1. a`<br>  `1. b`                                                                                          |                                        | `<ol><li>...`                                                                  |
| Task Lists                     | `- [ ] text`<br>`- [x] text`                                                                                |                                        | `<input type="checkbox" checked /><label>...`                                  |
| Blockquote                     | `> Blockquote`                                                                                              |                                        | `<blockquote><p>...`                                                           |
| Callouts                       | `> [!info] title`<br>`> content`                                                                            | > [!:custom emoji:]                    |                                                                                |
| Fenced Code Blocks             | `` ```lang ``<br>`` ~~~lang ``<br>`\t...`                                                                   | No indent                              | `<pre class="..."><code>...`                                                   |
| Equation Block                 | `$$`                                                                                                        |                                        | `<math>`                                                                       |
| Footnotes                      | `[^1]`                                                                                                      |                                        |                                                                                |
| <h4>Interactive elements</h4>  |                                                                                                             | <br>                                   |                                                                                |
| Disclose                       |                                                                                                             | `:::summary`<br>`details`<br>`:::`<br> | `<details><summary>...`<br>`...`                                               |
| Popover                        |                                                                                                             |                                        | Popover API                                                                    |
| Forms                          |                                                                                                             |                                        | `<form>` Form API                                                              |
| <h4>Image and multimedia</h4>  |                                                                                                             |                                        |                                                                                |
| Images                         | `![alt](src)`                                                                                               |                                        | `<img src="..." alt="..."/>`                                                   |
| Videos                         | `![](src)`                                                                                                  |                                        | `<video><source src="..." />`                                                  |
| Audio                          |                                                                                                             |                                        | `<audio controls src="..."></audio>`                                           |
| Figures                        |                                                                                                             |                                        | `<figure>{media}<figcaption>...`                                               |
| Files                          |                                                                                                             |                                        |                                                                                |
| <h4>Simple Tables</h4>         | `\| foo \| bar \|`<br>`\| --- \| --- \|`<br>`\| baz \| bim \|`                                              |                                        | `<table>`<br>  `<thead><tr><th>...`<br>  `<tbody><tr><th>...`                  |
| <h4>Demarcating <br>edits</h4> |                                                                                                             |                                        |                                                                                |
| <ins>Insertion</ins>           |                                                                                                             | `{++...++}`                            | `<ins>`                                                                        |
| Deletion                       |                                                                                                             | `{--...--}`                            | `<del>`                                                                        |
